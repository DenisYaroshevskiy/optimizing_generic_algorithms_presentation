merge

What affects perfromance of a merge?

  • Total size
  • How elements are mixed
  • Comparison
  • Non templated operations
  • Different iterator types
  • Cache locality of data
  • Binary size

What did we affect?

  • Non templated operations
  • Binary size (+ 11 instructions ~20%)

Basic merge benchmark

  • Merge 2000 integers
  • Parametarized on 2 sizes (represent distribution)
  • Double-check on int64 and doubles

What did we affect?

  • Extra comparison/assignment
  • Some iterator depending operations
  • Non templated operations
  • Binary size for ints (+ 11 instructions ~20%)
